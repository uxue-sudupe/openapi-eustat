Documentación de la API de Eustat

Introducción

El servicio API de Eustat permite acceder mediante peticiones URL a toda la información disponible en su banco de datos. Las peticiones URL tienen la siguiente estructura:

https://www.eustat.eus/bankupx/api/v1/{lang}/DB/[id]

Parámetros de la URL

El campo {lang} se corresponde con el idioma. Es obligatorio y puede tomar los siguientes valores:

  • ES: español
  • EU: euskera
  • EN: inglés

El campo [id] es opcional y corresponde al identificador de la tabla con los datos o metadatos de interés:

  • Si no se incluye en la consulta, se obtiene el catálogo de tablas del banco de datos.
  • Si se incluye el id y se usa la función GET, se obtienen los metadatos de la tabla.
  • Si se incluye el id y se usa la función POST junto con la consulta en formato JSON, se obtienen los datos de la tabla.

Formato de consulta JSON

Para formular las consultas en formato JSON, se puede usar la interfaz de usuario del banco de datos, y seleccionar las variables y valores de interés en la página correspondiente a la tabla.

Haga clic en el apartado “Disponer de esta tabla en su aplicación”. Esto le brindará información sobre la URL y la consulta que debe enviar para recoger los datos con la API.

API interactiva

La API interactiva de Eustat permite explorar y probar las funcionalidades de la API directamente desde una interfaz gráfica basada en Swagger. A través de esta herramienta, los usuarios pueden:

  • Realizar peticiones GET y POST para consultar datos.
  • Probar diferentes configuraciones de parámetros y filtros.
  • Visualizar la respuesta de la API en tiempo real y descargar los datos en formato json.
